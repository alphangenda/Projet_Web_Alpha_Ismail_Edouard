Déverrouillez

Approchez-vous de la trottinette de votre choix.

Trois manières de la déverrouiller :

  • Entrer le code de votre carte membre à 5 chiffres sur l’interface
  • Scanner votre carte membre ajoutée à votre compte
  • Scanner le QR code via l’application mobile

Sécurité : La trottinette est verrouillée tant qu’elle n’est pas déverrouillée.

  • Si elle est déplacée sans déverrouillage → un signal sonore se déclenche.
  • Si quelqu’un tente de rouler verrouillée → bruit d’alerte immédiat.

  • Roulez prudemment, respectez le code de la route et portez un casque.
  • Règles de sécurité
  • Privilégiez les pistes cyclables et les voies partagées.

Vous avez un arrêt : utilisez la béquille pour stationner correctement la trottinette.
